# 輪郭ラボ

> 世界に輪郭を。

輪郭ラボは、世の中で「分かった気になっている概念」を、
定義・境界・制約で明確化し、再利用できる思考の型として提供するメディアです。

単なる解説ではなく、次の4点まで落とし込みます。

1. それは何か／何ではないか（境界）
2. 成り立つ条件（制約）
3. 似た概念との切り分け
4. その結果として何が判断・予測できるようになるか

狙いは「知識」を増やすことではなく、読者が実務で使える「思考ツール」を渡すことです。

## 対象読者

概念を道具として使いたい準上級者以上を想定しています。

- 技術
- AI
- 制度
- 経済
- その他、複数領域を横断するテーマ

## 編集方針

- 言葉の定義を先に置く
- 適用範囲と限界を明示する
- 類似概念との差分を検証する
- 判断可能性・予測可能性に接続する
